Sharing code is a way to learn from others and let the other learn from you. And when time comes to share your work, chances are that you will want to share code that you feel proud about.

Update your server and make it serve an html page with a link to your source repository.
When accessing your repository, one should should read a welcome message referencing this game.

The id of the link should be #repository-link.
The repository home should contain a #readme element containing the text 'YoseTheGame'

Note: github and bitbucket both display, in a #readme element, the content of a readme file present in the root folder of a repository.
