Power of two

In mathematics, a power of two means a number of the form 2*exp(n) where n is an integer, i.e. the result of exponentiation with as base the number 2 and as exponent the integer n.

You're expected to provide an url that should accept a GET request with a power of 2 as parameter and should respond a JSON containing the decomposition of this number

For example, receiving

http://your/url?number=16

You should respond

{
    "number" : 16,
    "decomposition" : [ 2, 2, 2, 2 ]
}

Take it!

Your server will be called with a random power of two.

Expected
{ some: thing }
Got:
{ other: one }
