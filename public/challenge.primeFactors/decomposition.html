Nice ! I guess you can't wait to show that you can do more. Here it is: the decomposition challenge, to explore the decomposition of any number in its prime factors.

Update your server to make it accept a GET request with a positive integer parameter and should respond a JSON containing the prime factors decomposition of this number

For example, receiving

http://your/server?number=300

You should respond

{
    "number" : 300,
    "decomposition" : [ 2, 2, 3, 5, 5 ]
}

Take it!

Your server will be called with a random power of two.

AND the previous challenge should still pass :)
