Now that your server can decompose power of two numbers, you might feel a little nervous about all the hackers around that will eventually try to crash down your wonderful code.

Update your server to make it accept a GET request with a string as parameter and make it respond a JSON containing the 'not a number' error message.

For example, receiving

http://your/server?number=hello

You should respond

{
    "number" : "hello",
    "error" : "not a number"
}

Take it!

Your server will be called with a string.

AND the previous challenges of the level should still pass :)
