Imagine you are a fire-fighting plane pilot...

When you receive

http://your-server/fire/geek?width=3&map=...P...WF

You should respond the following Json

{
    map: [
        "...",
	"P..",
	".WF"
    ],
    moves: [
        { dx: 0, dy: 1 },
        { dx: 1, dy: 0 },
        { dx: 1, dy: 0 },
    ]
}

OR

{
    map: [
        "...",
	"P..",
	".WF"
    ],
    moves: [
        { dx: 1, dy: 0 },
        { dx: 0, dy: 1 },
        { dx: 1, dy: 0 },
    ]
}

OR

Any moves that will lead your plane (P) to the fire (F), passing first above the water (W)
